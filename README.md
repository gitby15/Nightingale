
Nightingale
========
######React studying project


Goal
-----
>1. Build, run, and test automatically.
>2. System modularization, and every component is reusable.
>3. 100% test coverage

Online Demostrate
------
http://lijin.leanapp.cn


Tech stack
-----

> React + Redux + Webpack + Karma


How to run
-----
>1. install nodejs from https://nodejs.org
>2. clone code
>3. cd Nightingale
>4. npm install
>5. npm start
